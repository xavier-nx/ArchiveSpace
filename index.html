Características Principales

Autenticación Segura

Soporte para autenticación local y LDAP con gestión de roles y permisos por repositorio.

Gestión Archivística

Importación/Exportación

Sobre el Proyecto

Objetivo General

Construir y aplicar un sistema archivístico con arquitectura sólida para gestionar recursos, accesiones, agentes, sujetos y objetos digitales.

Sprint 1 - Autenticación y Gestión de Usuarios

Duración: 1 semana | Objetivo: Establecer la base de seguridad y control de acceso del sistema.

  • Configuración inicial del proyecto
  • Implementación de autenticación local
  • Implementación de autenticación LDAP
  • Gestión de usuarios y roles
  • Configuración externa del sistema
